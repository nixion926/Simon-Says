Simon Says

Press Any Key to Start Game!

Rules:

1. Simon will give the first signal.

2. Repeat the signal by pressing the same color button.

3. Simon will duplicate the first signal and add a new signal.

4. Repeat these two signals by pressing the same color buttons, in order.

5. Continue playing as long as you can repeat each sequence of signals correctly.

6. If you fail to repeat a sequence exactly, the game is over.
